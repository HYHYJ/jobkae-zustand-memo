# Zustand Memo App

## 개발환경 / 실행방법
### `Node.js version`
16.19.0

### `yarn start`
yarn start 커맨드를 입력하면 실행됩니다

### `yarn build`
yarn build 커맨드를 입력하면 빌드를 진행합니다
